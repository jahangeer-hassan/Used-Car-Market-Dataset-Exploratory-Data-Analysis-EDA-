# 🚗 Used Car Market Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on a Used Car Market dataset using **Python**. The goal of this analysis is to discover hidden patterns, understand market trends, and identify the factors that affect used car prices.

Through data cleaning, visualization, and statistical analysis, this project provides meaningful insights into the used car industry.

---

# 🎯 Objectives

* Understand the structure of the dataset
* Analyze factors affecting car prices
* Identify trends in fuel type and transmission
* Compare mileage, kilometers driven, and resale value
* Visualize data using graphs and charts
* Perform univariate and bivariate analysis

---

# 🛠️ Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

# 📂 Dataset Features

The dataset contains important information about used cars such as:

* Car Name

* Selling Price
* Fuel Type
* Transmission
* Mileage
* Kilometers Driven
* Owner Type
* Engine Capacity
* Seats

# 🔍 Key Insights

* Newer cars generally have higher selling prices.
* Cars with lower kilometers driven have better resale value.
* Automatic transmission cars tend to be more expensive.
* Fuel type impacts market demand and pricing.
* Premium brands maintain stronger resale prices.
